Dual nameserver setup

  • Set the external one up to be as restrictive as possible. Two config files.

  • Set the internal one as a caching-only nameserver

    • Default configuration, usually.

  • Set them up to only work on certain interfaces.

    • listen-on in options
    • listen-on { <ip-address>; <another ip address>;};

  • Then start up named, but with different conf files

    • named -c conf-file
